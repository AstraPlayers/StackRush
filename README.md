# StackRush 🎮

Un jeu HTML rapide et addictif où tu dois empiler les blocs parfaitement pour obtenir le meilleur score possible.

## 🌍 Jouer au jeu
👉 https://astraplayers.github.io/StackRush

---

# ✨ Fonctionnalités

- 🎯 Gameplay simple et addictif
- 📱 Compatible mobile et PC
- ⚡ Interface rapide
- 🧱 Système de stacking fluide
- 🏆 Score en temps réel

---

# 🕹️ Comment jouer

1. Lance le jeu
2. Clique ou appuie sur l’écran
3. Empile les blocs au bon moment
4. Essaie d’obtenir le meilleur score possible
